使い方と1日100語ルーチン

結論：毎日100語、意味OFF、3秒は絶対判定、復習は後回し。使い方・守ること・週次運用はこのタブだけ見ればOKです。

週次ログ出力メモ

毎週 日曜または月曜の夜 に、学習ログを JSON または CSV で出力して、このチャットへアップロードします。

  • おすすめ：JSON出力（分析用に最適）
  • 補助：CSV出力（表で見る用）
  • 目安：毎週 日曜・月曜 20:30
  • アップロード後、週次サマリー・苦手単語・改善/悪化・次週方針を作成

毎日のやり方

  1. 全体100 または Part1 100 を選ぶ。
  2. 意味表示OFF で始める。
  3. 3秒以内に 知ってる / 知らない を押す。
  4. 迷ったら 知らない。
  5. 終わるまで調べない。復習しない。

曜日メニュー目的
月Part1 100基礎反応
火Part1 100基礎定着
水Part1+2 100範囲拡張
木全体100全体接触
金弱点100unknown / timeout対策
土全体100ランダム耐性
日復習・KPI確認調整日

守ること

  • 1問で止まらない。
  • 正確な和訳を探しにいかない。
  • 3秒超えは負けではなくデータ。
  • 「知ってる」は3秒以内に意味・イメージが出た時だけ。
  • 100語で終える。やりすぎない。

さらに効果的な使い方

  • 朝・夜など時間帯を変えて、効果最大化時間帯を見る。
  • 最初の2週間は同じルーチンで回して、データを安定させる。
  • 2週間後から、余裕日だけ200語に増やす。
  • 復習タブでは unknown / timeout / 直近悪化 を優先する。
  • 管理タブでは 接触単語数 / timeout率 / 直近3回のms だけ見る。

効果が出ているサイン

  • timeoutが減る。
  • 直近3回の反応速度が短くなる。
  • unknownがknownに変わる。
  • Partをまたいでも反応が落ちにくくなる。

解放ルール

  • 1日のattempt数が100以上で、その日を「達成」とします。
  • 達成日が1日増えるごとに、左上から1ピースずつ解放します。
  • 30日達成で1枚完成です。
  • unknownやtimeoutがあってもOK。大事なのは毎日接触すること。
